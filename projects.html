Projects

Visit my GitHub to see my latest projects: http://www.github.com/jmmiller7

Trendy – An app that brings together trending items from Giphy and Imgur. User can also search terms across both APIs.
– uses Glide, GiphyAPI, ImgurAPI and PagerSlidingTabStrip.

MyGolfBuddy – Android application that allows a user to keep track of their golf scores at a specified golf course. Currently in development. Future plans include tying into a website and implement OCR technology where the app can take a picture of and save a completed paper scorecard. The motivation behind this application is to learn about using an external database, web development and web services with an android application. See the GitHub page for latest updates.

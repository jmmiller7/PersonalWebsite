Learning Content

Thanks to the world wide web and many published books, there’s an endless supply of learning material. The content listed below are things that I’ve found useful on my quest to becoming a successful software developer.

Websites and Tutorials

Derek Banas on YouTube – Derek Banas provides a broad coverage of programming languages and techniques. Whether it be learning Object-Oriented programming with Java or C#, or Web Development with HTML, CSS and JavaScript, you are guaranteed to gain some knowledge from this page. Also visit Derek’s website, www.newthinktank.com.
Codecademy – The Codecademy website provides many hands-on tutorials that walk you through learning the syntax of a language, and how to effectively apply what you learn.

Books

Android: How to Program – This book is a wonderful introduction to the world of Android app development! It begins by introducing the developer to the Android OS as well as the components within the IDE. Although this book is based on the use of Eclipse with the Android SDK, it is easy to translate the coverage to Android Studio. This book has an interactive feel to it, in which it walks you through the creation of a fully-functional Android app in each chapter.
Head First Design Patterns – If anybody is looking to improve their knowledge on software design patterns, this is the place to look! Instead of the traditional “text and figure” textbook, the authors have taken a unique and indirect approach in which they come up with an animation of a scenario, and then apply that scenario to a design pattern.
